# Student Registration System

A full-stack web application for managing student registrations and profiles. The project is built using Spring Boot for the backend and HTML, CSS, and JavaScript for the frontend.

## Features

* Student registration
* User login and authentication
* Profile management
* Update student information
* Delete student records
* View all registered students
* Form validation

## Technologies Used

### Backend

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* REST API

### Frontend

* HTML
* CSS
* JavaScript

## Project Status

🚧 Currently under development.

The frontend registration form is being developed using javascript , and the backend is being developed using Spring Boot.

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project in your IDE.

3. Configure MySQL database settings in `application.properties`.

4. Run the Spring Boot application:

   ```bash
   mvn spring-boot:run
   ```

5. Open the application in your browser.
